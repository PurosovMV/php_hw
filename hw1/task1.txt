<?php
    $a = 5;
    $b = '05';

    var_dump($a == $b);         // Почему true?

    $a имеет целочисленный тип данных, $b строковый тип, но при сравнении переменных PHP приведёт переменную $b к целочисленной var_dump ((int) $b) 
  

    var_dump((int)'012345');     // Почему 12345?

    Преобразование в целое число с явным указанием типа, отбрасывается ноль и в результате  формируется число
    с первого значения не равного 0 или NULL.

    var_dump((float)123.0 === (int)123.0); // Почему false?

    Явное указание типов значений + строгое равенство, float - дробное число, int - целое число 123.0 не равно 123

    var_dump(0 === 'hello, world'); // Почему false?

    Строгое равенство, но при этом  типы данных не совпадают следовательно false.
